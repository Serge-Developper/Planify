# 📌 Planify

## 🧠 Présentation du projet

Planify est une application de gestion et d’organisation conçue pour aider les utilisateurs à mieux planifier leurs tâches, leurs événements et leur emploi du temps de manière simple, rapide et efficace, en intégrant un système de gamification.

Grâce à des cosmétiques, des récompenses, des coins, des factions et une boutique quotidienne, l’application transforme l’organisation en une expérience ludique visant à booster la productivité et l’engagement des étudiants en BUT MMI dans leurs devoirs et examens.

L’objectif du projet est de centraliser la planification dans une interface intuitive afin d’améliorer la productivité au quotidien.

---

## 🚀 Fonctionnalités

* 📅 Création et gestion de tâches
* 🕒 Planification d’événements et rappels
* 👥 Gestion par groupes (promotions / classes / équipes)
* 🔔 Système de notifications
* 📊 Organisation claire et structurée des données

## 📦 Installation

```bash
git clone https://github.com/ton-repo/planify.git
cd planify
npm install
npm start
```

---

## 📱 Utilisation

Une fois l’application lancée :

1. Créer un compte ou se connecter (seul l’administrateur peut créer de nouveaux comptes)
2. Ajouter des tâches ou événements
3. Organiser selon les groupes ou catégories
4. Suivre son planning en temps réel
